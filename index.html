Compare to VPS

Cloud VPS nodes (e.g. $40–80/mo for 500 GB+) often cost 2–5× more annually than a home Pi setup by 2024. Self-hosting wins on long-term hodler timelines, especially with pruned mode for non-archival use. Trade-off: you manage uptime, security patches, and power outages.

Why It Got Cheaper — FAQ
HDD price deflation: $/TB fell from ~$110 (2009) to ~$10 (2026) — a 90%+ drop in storage cost per gigabyte.
Efficient hardware: Raspberry Pi 4/5 runs Bitcoin Core with 4–8 GB RAM, replacing $500+ desktops from 2009–2012.
Bandwidth abundance: Unlimited home internet became standard; node traffic is a tiny fraction of monthly caps after initial sync.
Software optimization: Pruning, compact blocks, and better I/O reduced resource needs without sacrificing network participation.
The catch: Absolute chain size still grows ~50–60 GB/year. Archival nodes always need more disk — but disk got cheaper faster than the chain grew.
